# XMR

Monero 是所有加密货币中隐私性最强、默认匿名的币种之一。

## Monero(XMR)

* [https://www.getmonero.org/](https://www.getmonero.org/)
* [https://monerohash.com/](https://monerohash.com/)

| 项目     | 内容                                       |
| -------- | ------------------------------------------ |
| 名称     | Monero                                     |
| 代币符号 | XMR                                        |
| 上线时间 | 2014 年 4 月                               |
| 官方网站 | [getmonero.org](https://www.getmonero.org) |
| 区块时间 | \~2 分钟                                   |
| 供应总量 | 无硬性上限（有 tail emission 机制）        |
| 共识机制 | Proof of Work（工作量证明）                |
| 算法     | RandomX（CPU 优化，抗 ASIC）               |
| 匿名特性 | 默认启用，不能选择关闭                     |


| 技术                  | 功能说明                  |
| ------------------- | --------------------- |
| **Ring Signature**  | 隐藏谁花的钱（混入假输入）         |
| **Stealth Address** | 隐藏收款地址（每次都是临时一次性隐蔽地址） |
| **RingCT**          | 隐藏交易金额（环机密交易，金额加密）    |
| **Key Image**       | 防止双花，但不能反推出用户身份       |
| **Bulletproofs**    | 零知识证明，用于高效验证金额范围      |
