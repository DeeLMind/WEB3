# NEAR

> 官网：https://near.org  
> NEAR 是一个专注于开发者体验、用户可用性和高性能的分片公链平台。

---

## 🧱 一、NEAR 是什么？

NEAR 是一个基于分片（sharding）的第 1 层区块链，旨在提供可扩展、低延迟且开发者友好的智能合约平台。它解决了传统区块链在**性能、可用性和成本**上的痛点。

- **共识机制**：基于 Nightshade + Doomslug 的改进版 PoS
- **智能合约语言**：Rust、AssemblyScript（类 TypeScript）
- **目标用户**：Web3 开发者、应用构建者、大规模用户应用

---

## 🚀 二、NEAR 核心特性

### 1. 🌐 分片架构（Nightshade）
- 支持并行处理多个分片中的交易，**高可扩展性**
- 最终目标：支持数十亿用户

### 2. ⚡ 高性能
- 快速出块（约每 1 秒一个区块）
- 高 TPS（理论上可达数万笔/秒）
- **低 gas 费用**，用户体验类似 Web2

### 3. 🤝 强用户友好性
- 支持**人类可读账户名**（如 `alice.near`）
- 钱包内置社交账户恢复（邮箱、手机号）
- 一键登录 & 授权 dApp，**降低用户门槛**

### 4. 🧑‍💻 面向开发者
- 提供 Rust + TypeScript SDK，**低门槛开发体验**
- 丰富的 Dev Tool：NEAR CLI、Testnet、Sandbox、Explorer、Faucet
- 强大的文档：https://docs.near.org/

### 5. 🔁 互操作性
- 支持通过 Rainbow Bridge 与以太坊交互
- 支持 EVM 合约部署（Aurora 项目）

---

## 🌉 三、生态系统组成

| 模块 | 说明 |
|------|------|
| 🌈 Rainbow Bridge | NEAR 与 Ethereum 的桥接方案，支持资产双向转移 |
| 🧬 Aurora | 在 NEAR 上运行的 EVM 兼容网络，支持 Solidity 合约 |
| 🧾 NEAR Wallet | 官方钱包，支持账户恢复、多签、NFT |
| 🌱 NEAR Foundation | 推动生态发展和资金支持的非营利组织 |
| 🚀 Pagoda（前 NEAR Inc） | 核心协议开发团队和平台服务 |

## 📊 五、NEAR vs 其他公链对比

| 特性 | NEAR | Ethereum | Solana | Polkadot |
|------|------|----------|--------|----------|
| 分片机制 | ✅ Nightshade | ❌ | ❌（单链） | ✅（异构多链） |
| 出块时间 | ~1s | ~12s | ~0.4s | ~6s |
| 开发语言 | Rust / TS | Solidity / Vyper | Rust / C | Rust |
| 账户系统 | 人类可读 | 地址为哈希 | 地址为哈希 | 地址为哈希 |
| 互操作性 | EVM + Bridge | EVM native | 桥接需开发 | XCMP |
| 用户体验 | 强 | 弱 | 中等 | 技术性强 |
